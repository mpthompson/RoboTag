This directory is included because it provides very useful example code for 
accessing a video stream using the DirectShow APIs.  This was updated to work
with Microsoft Visual C++ 2008 Express Edition and the Studio 2008 and
the Microsoft DirectX SDK (March 2008).  The after compiling the code 
the following command should display the camera image on a OpenGL surface:

C:\...>glutSampled WDM_camera.xml
